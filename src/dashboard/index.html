Tính năng thử nghiệm

Các tối ưu chưa ổn định; nếu phát hiện bất thường có thể tắt bất cứ lúc nào
Tái sử dụng hội thoại Cascade
Khi hội thoại nhiều lượt, tái sử dụng cùng một cascade_id, chỉ gửi message user mới nhất tới Windsurf, để server giữ cache ngữ cảnh. Khi hit cache giảm TTFB và lưu lượng upload đáng kể; nếu miss hoặc account/LS đổi sẽ tự rollback về phiên mới. Yêu cầu client giữ đủ lịch sử và append đúng thứ tự(ví dụ new-api, OpenWebUI).
Bật tái sử dụng hội thoại Cascade
Mặc định tắt. Bật xong sẽ áp dụng ngay cho pool hội thoại hiện tại.
Inject identity model
Khi bật, inject một system prompt vào ngữ cảnh mỗi request để model tự xưng theo identity được yêu cầu (ví dụ khi gọi claude-opus-4.6 thì model sẽ tự nói nó là Claude Opus 4.6, do Anthropic phát triển). Ghi đè identity Windsurf mặc định của Cascade.
Bật inject identity model
Mặc định bật. Hỗ trợ model của Anthropic / OpenAI / Google / DeepSeek / xAI...
